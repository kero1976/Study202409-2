初期パスワードは端末 (ターミナル) に表示された以下の部分です。

「temporary password is generated for root@localhost: <これ>」
初期パスワードを変更しないと使えないので、以下のコマンドで変更します。

Copy
SET PASSWORD = '大文字・小文字・数字・記号のすべてを含む好きなパスワード';
